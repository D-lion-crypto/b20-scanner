# Security Policy

## Secrets

Never commit:

- CDP API Key IDs or Secrets
- CDP JSON key files
- private RPC URLs
- wallet private keys or seed phrases
- `.env` files

If a key was pasted into chat, screenshots, logs, or a public repository, revoke it immediately and create a new one.

## Reporting

Open a GitHub issue for non-sensitive bugs. For sensitive security issues, contact the repository owner privately.
